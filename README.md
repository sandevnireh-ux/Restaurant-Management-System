#                            **Restaurant-Management-System**
# Project Overview
-- A complete Java Swing based Restaurant Management System implementing basic OOP concepts with a user friendly GUI interface.
# Features
-- Dual Login System: User & Admin
-- Interactive Menu
-- Shopping Cart
-- Payment Processing: Cash & Card payments
-- Admin Dashboard: Staff details & Financial reports
-- Final Receipt Generation
# Technologies Used
-- Java
-- Java Swing
-- OOP Principles
-- Framework Collections (ArrayList)
# Project Structure
-- The implemented OOP concepts:
1. Encapsulation - Private fields with public getters
2. Abstraction - Complex GUI behind simple methods
3. Inheritance - Swing component hierachy
4. Polymorphism - Method overriding in event handlers
5. Composition - Complex objects from simpler components
